#Movie_Ticket_Booking
Movie_ticket_booking used teq MongoDB, Expressjs, Nodejs

![image](https://user-images.githubusercontent.com/73506478/215267352-1e46627e-18ae-4873-bdd0-28ec77df1afd.png)

## Features!

- **GET:** movies / movie
- **GET:** movies / location
- **GET:** movies / show
- **POST:** movies / movie
- **POST:** movies / theater
- **POST:** movies / show
- **POST:** movies / ticket
- **PUT:** movies / ticket/:id
